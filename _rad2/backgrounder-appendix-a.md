## New elements/rules

The following is a list of RAD2 elements or rules for which there was no provision in RAD.

| &nbsp; | &nbsp; | 
| --- | --- |
| 4.1 | Reference code | 
| 4.2 | Level of description | 
| 6.3 | Technical access | 
| 7.3 | Appraisal/scheduling/destruction | 
| 8.4 | Publication note | 
| 11 | Description control area | 
| 13.B23-25 | Access points for collections | 
| 15.2 | Relationships between entities and archival materials | 
| 15.3 | Relationships between entities and materials not housed in an archival repository | 
| 19 | Authority record control area | 

## Omitted elements

The following is a list of RAD elements that do not appear in RAD2.

| &nbsp; | &nbsp; | 
| --- | --- |
| 1.1C | General material designation | 
| 1.5E | Accompanying material | 
| 1.8B9c | Accompanying material | 

## Expanded elements

The following elements represent expanded treatment or broadened scope.

| RAD2 | Description | RAD | Description | 
| --- | --- | --- | --- |
| 4.4 | Dates | 1.4B | Dates of creation | 
| 5.2 | System of arrangement | 1.7D, 1.8B13 | Scope and content, Arrangement note | 
| 6.4 | Conditions governing reproduction | 1.8B16c, 1.8B16d | Terms governing use and reproduction, Terms governing publication | 
| 8.2 | Existence/location of copies | 1.8B15b | Availability of other formats | 

## Compressed elements

The following RAD2 elements bring together separate RAD elements.

| RAD2 | Description | RAD | Description | 
| --- | --- | --- | --- |
| 8.3 | Related units of description | 1.8B18, 1.8B20/a | Associated material, Related groups of records | 
| 9.3 | Mathematical data | 1.3, x.3 | Class of materials specific details |

| 7.4 | Accruals | 1.8B19 | Accruals |

| 8.2 | Existence and location of copies | 1.8B15b Availability of other formats | 
| 8.3 | Related material in the same institution | 1.8B20 | Related groups of records in different fonds |
